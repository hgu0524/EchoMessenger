(C# 코딩) 에코 메신저

##개요

* C# 프로그래밍 학습을 위한 Windows Forms 기반 프로젝트

* 1줄 소개: 사용자 키보드 입력을 받아 메시지를 출력하고 관리하는 프로그램

* 사용한 플랫폼: C#, .NET Windows Forms, Visual Studio, GitHub

* 사용한 컨트롤:

* Label, TextBox, ListBox, Button

* 사용한 기술과 구현한 기능:

* Visual Studio를 이용한 UI 디자인 구성

* string 클래스를 활용한 사용자 입력 데이터 처리

* DateTime 클래스를 이용한 현재 시간 정보 출력

* ListBox를 활용한 메시지 저장 및 관리 기능 구현

## 실행화면 (과제1)

* Label, TextBox, Button, ListBox를 이용한 기본 UI 구성

* 전송 버튼 클릭 시 입력된 메시지를 리스트에 추가

* 메시지 전송 후 입력창 초기화

* 구현 내용과 기능 설명

* 메시지는 한 줄씩 누적되어 대화 형태로 표시된다.

* 메시지가 많아지면 스크롤을 통해 이전 메시지를 확인할 수 있다.

##실행 화면 (과제2)




* 전송 후 입력창 자동 초기화 및 포커스 이동

* Enter 키를 이용한 메시지 전송 기능 구현

* 공백 입력 방지 기능 추가

* 구현 내용과 기능 설명

* Enter 키 입력으로 버튼 클릭 없이 메시지를 전송할 수 있다.

* 메시지 전송 후 자동으로 입력창에 커서가 위치하여 연속 입력이 가능하다.

* 공백 또는 빈 문자열 입력 시 메시지가 추가되지 않도록 처리하였다.

---

##실행 화면 (과제3)




* 메시지에 현재 시간 추가

* 메시지 개수 표시 기능 구현

* 문자열 공백 제거

* 구현 내용과 기능 설명

* 메시지 전송 시 현재 시간을 HH:mm:ss 형식으로 함께 출력하였다.

* ListBox에 저장된 메시지 개수를 계산하여 Label에 표시하였다.

* Trim()을 사용하여 입력 문자열의 앞뒤 공백을 제거하였다.

---

##실행 화면 (과제4)




* 선택한 메시지 삭제 기능 구현

* 전체 메시지 삭제 기능 구현

* 입력 메시지 글자 수 제한

* 구현 내용과 기능 설명

* ListBox에서 선택한 메시지를 삭제 버튼으로 제거할 수 있다.

* 전체 삭제 버튼을 통해 모든 메시지를 한 번에 초기화할 수 있다.

* 입력 메시지를 50자로 제한하고 초과 시 경고 메시지를 출력하도록 구현하였다.
